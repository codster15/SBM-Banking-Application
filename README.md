# SBM Bank Project

## Description
The **SBM Bank Project** is a web-based clone of SBM Bank's online banking platform. It allows users to manage their accounts, make transactions, pay bills, view loan details, and more, all within a secure and user-friendly interface.

## Features
- **User Registration & Login**: Secure user authentication and account management.
- **Dashboard**: View account balances, recent transactions, and quick links.
- **Fund Transfers**: Transfer funds between accounts or to external beneficiaries (IMPS, NEFT, RTGS).
- **Bill Payments**: Pay utilities like electricity, water, and internet.
- **Loan Management**: View and manage loan balances, EMI schedules, and interest rates.
- **Transaction History**: View detailed transaction logs and export statements.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript, Bootstrap, Thymeleaf
- **Backend**: Spring Boot, Spring Security, Spring Data JPA
